# RD-11 - Tarifas ativas e catálogos controlados

**Descrição:**
Cada cálculo financeiro deve usar a tarifa diária ativa associada ao tipo de alojamento e os tipos de serviço extra definidos pela direção em catálogo controlado. Os pagamentos devem persistir os valores efetivamente calculados para preservar a leitura histórica de reservas, estadias e faturação.

**Origem:** US-06

**Prioridade:** Must Have
