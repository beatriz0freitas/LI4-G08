# RF-01 - Dashboard operacional

**Descrição:**
O sistema deve disponibilizar um dashboard acessível ao perfil de diretor contendo, no mínimo, os seguintes indicadores: taxa de ocupação atual (percentagem de boxes ocupadas), número de estadias ativas, número de reservas futuras e valor total de faturação diária e mensal. Os dados devem ser atualizados automaticamente com uma frequência máxima de 60 segundos ou sempre que ocorra um evento relevante (check-in, check-out, criação ou cancelamento de reserva, registo de pagamento). O dashboard deve permitir a visualização gráfica e filtragem por período temporal.

**Origem:** US-01, US-02

**Prioridade:** Must Have
