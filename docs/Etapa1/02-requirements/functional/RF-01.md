# RF-01 - Dashboard operacional

**Descrição:**
O sistema deve disponibilizar um dashboard com indicadores de ocupação
em tempo real e de faturação, consultável pelo diretor.

**Refinamento:**
O sistema deve disponibilizar ao perfil de diretor uma visão operacional com taxa de ocupação atual, número de estadias ativas, número de reservas futuras, faturação total, faturação do mês corrente, pagamentos pendentes e estado de limpeza dos alojamentos. A filtragem por período e agrupamento detalhado pertence aos relatórios operacionais e financeiros definidos em RF-03.

**Origem:** US-01, US-02

**Prioridade:** Must Have
