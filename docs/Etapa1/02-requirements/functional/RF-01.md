# RF-01 - Dashboard operacional

**Descrição:**
O sistema deve disponibilizar um dashboard com indicadores de ocupação
em tempo real e de faturação, consultável pelo diretor.

**Refinamento:**
O sistema deve disponibilizar ao perfil de diretor uma visão operacional com taxa de ocupação atual, número de estadias ativas, número de reservas futuras, faturação diária e mensal e pagamentos pendentes, permitindo filtragem por período temporal. Os indicadores devem refletir reservas, estadias, pagamentos, estado de limpeza e faturação complementar registada no check-out, com atualização em tempo útil para apoiar a decisão operacional.

**Origem:** US-01, US-02

**Prioridade:** Must Have
