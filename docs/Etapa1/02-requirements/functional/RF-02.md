# RF-02 - Gestão de perfis de acesso

**Descrição:**
O sistema deve permitir gerir colaboradores e respetivos perfis de acesso, incluindo criação, edição e desativação de contas.

**Refinamento:**
O sistema deve permitir gerir colaboradores e respetivos perfis de acesso, incluindo criação, edição e desativação de contas. Deve distinguir, pelo menos, os perfis de diretor, funcionário de receção, cuidador, médico veterinário e responsável pela limpeza, garantindo que alterações de perfil ou desativação não eliminam a rastreabilidade das ações já realizadas por colaboradores.

**Origem:** US-03

**Prioridade:** Must Have
