# RF-02 - Gestão de perfis de acesso

**Descrição:**
O sistema deve permitir gerir os perfis de acesso dos colaboradores, incluindo criação, edição e remoção de permissões por perfil.

**Refinamento:**
O sistema deve permitir gerir os perfis de acesso dos colaboradores, incluindo criação, edição e remoção de permissões por perfil. Deve distinguir, pelo menos, os perfis de diretor, funcionário de receção, cuidador, médico veterinário e responsável pela limpeza, garantindo que alterações de permissões não eliminam a rastreabilidade das ações já realizadas por colaboradores.

**Origem:** US-03

**Prioridade:** Must Have
