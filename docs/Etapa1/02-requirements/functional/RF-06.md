# RF-06 - Controlo de disponibilidade de alojamentos

**Descrição:**
O sistema deve controlar a disponibilidade dos alojamentos em tempo real,
impedindo a criação de reservas em conflito ou para alojamentos não preparados.

**Refinamento:**
O sistema deve determinar a disponibilidade de cada box em tempo real e considerar como indisponível qualquer alojamento com reserva sobreposta ao período pretendido, com estadia ativa nesse período, com limpeza diferente de concluída ou incompatível com a espécie ou capacidade necessária. O sistema deve impedir a criação de reservas que violem estas condições.

**Origem:** US-08, US-14, US-23

**Prioridade:** Must Have
