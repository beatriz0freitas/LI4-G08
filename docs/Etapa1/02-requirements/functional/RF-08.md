# RF-08 - Check-in e pagamento de estadia

**Descrição:**
O sistema deve suportar o processo de check-in, registando a data de entrada, o alojamento atribuído e os dados relevantes do animal, e processar o pagamento da estadia nesse momento.

**Refinamento:**
O sistema deve suportar o processo de check-in, validando que existe uma reserva associada, registando a data de entrada, abrindo a estadia do animal, impedindo que o mesmo animal tenha outra estadia ativa e processando o pagamento base da estadia. O valor base deve ser calculado a partir da duração reservada e da tarifa diária do tipo de alojamento.

**Origem:** US-09, US-12

**Prioridade:** Must Have
