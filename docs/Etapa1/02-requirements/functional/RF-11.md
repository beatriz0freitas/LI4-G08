# RF-11 - Plano de cuidados

**Descrição:**
O sistema deve disponibilizar o plano de cuidados de cada animal em estadia, consultável por qualquer cuidador.

**Refinamento:**
O sistema deve disponibilizar, durante a estadia, o plano de cuidados ativo de cada animal a cuidadores e perfis autorizados. O plano deve refletir cuidados recorrentes do animal, instruções da reserva e ajustes registados durante a estadia, incluindo tarefas, instruções livres e indicação de prioridade.

**Origem:** US-16, US-18

**Prioridade:** Must Have
