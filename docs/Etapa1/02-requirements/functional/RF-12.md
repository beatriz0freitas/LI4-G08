# RF-12 - Registo de cuidados prestados

**Descrição:**

O sistema deve permitir o registo dos cuidados prestados a cada animal,
com identificação do cuidador, data e hora.

**Refinamento:**
O sistema deve permitir o registo de cada cuidado prestado a um animal em estadia ativa, incluindo data e hora de realização, identificação do autor quando disponível e descrição/observações do cuidado. O acesso ao registo deve ser limitado a perfis autorizados pela configuração de segurança, e cada registo deve ficar associado à estadia do animal, sendo consultável por colaboradores com permissão de acesso.

**Origem:** US-17

**Prioridade:** Must Have
