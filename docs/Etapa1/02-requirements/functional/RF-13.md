# RF-13 - Alterações ao estado de saúde

**Descrição:**
O sistema deve permitir registar alterações ao estado de saúde de um animal associadas a uma estadia, com descrição, data/hora e severidade. As alterações recentes devem estar disponíveis para consulta pelo médico veterinário; alterações críticas devem refletir-se na prioridade do plano de cuidados ativo quando este existir.


**Origem:** US-18, US-26

**Prioridade:** Must Have
