# RF-15 - Estado de limpeza dos alojamentos

**Descrição:**
O sistema deve permitir o registo e atualização do estado de limpeza de cada alojamento, garantindo que apenas alojamentos com estado "concluído" podem ser considerados disponíveis para nova reserva.

**Refinamento:**
O sistema deve permitir registar e atualizar o estado de limpeza de cada alojamento. Após o check-out, o alojamento associado deve ficar marcado como pendente de limpeza; o responsável pela limpeza deve conseguir consultar alojamentos pendentes e marcar cada alojamento como limpo. Apenas alojamentos com limpeza concluída podem ser apresentados como disponíveis para novas reservas.

**Origem:** US-22, US-23

**Prioridade:** Must Have
