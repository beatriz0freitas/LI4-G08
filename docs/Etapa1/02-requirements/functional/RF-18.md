# RF-18 - Gestão de tipos, tarifas e serviços extra

**Descrição:**
O sistema deve permitir à direção gerir os tipos de alojamento, as respetivas tarifas diárias e os tipos de serviços extra disponíveis.

**Refinamento:**
O sistema deve permitir listar, criar, editar, desativar e reativar tarifas por tipo de alojamento e tipos de serviço extra. O cálculo financeiro deve usar apenas tarifas ativas e o registo de serviços extra deve usar tipos ativos do catálogo.

**Origem:** US-06

**Prioridade:** Must Have
