# RF-19 - Auditoria de operações críticas

**Descrição:**
O sistema deve permitir guardar auditorias para consultar o registo das operações críticas realizadas, identificando o autor, o momento e o tipo de operação, incluindo reservas, check-in, check-out, pagamentos, serviços extra, intervenções clínicas, alterações de limpeza, relatórios e gestão de colaboradores.

**Origem:** US-07

**Prioridade:** Should Have
