# UC-01 - Autenticar no Sistema

- Use Case: Autenticar no Sistema
- Descricao: Permite a um colaborador aceder ao sistema com credenciais individuais, ficando sujeito ao perfil de acesso que lhe foi atribuido.
- Cenario: Um colaborador inicia a sessao para aceder as funcionalidades necessarias ao seu trabalho.
- Pre-condicao: O colaborador tem conta ativa e credenciais validas.
- Pos-condicao: Sessao iniciada com as permissoes do perfil atribuido.
- Fluxo Normal:
  1. Colaborador acede ao sistema.
  2. Colaborador introduz as suas credenciais.
  3. Sistema valida as credenciais.
  4. Sistema identifica o perfil do colaborador.
  5. Sistema disponibiliza as funcionalidades permitidas para esse perfil.
- Fluxo Alternativo:
  - [Credenciais invalidas] (passo 3)
    1. Sistema informa que as credenciais nao sao validas.
    2. Colaborador pode tentar novamente.
