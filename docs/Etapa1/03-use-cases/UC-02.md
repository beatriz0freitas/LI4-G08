# UC-02 - Consultar Disponibilidade de Alojamentos

- Use Case: Consultar Disponibilidade de Alojamentos
- Descricao: Permite consultar alojamentos disponiveis para um periodo, considerando reservas, estadias em curso, estado de limpeza e compatibilidade com o animal.
- Cenario: Um tutor contacta o hotel e o funcionario verifica se existe alojamento disponivel para as datas pretendidas.
- Pre-condicao: Utilizador autenticado; periodo pretendido conhecido.
- Pos-condicao: Lista de alojamentos disponiveis apresentada ou indisponibilidade comunicada.
- Fluxo Normal:
  1. Utilizador seleciona a consulta de disponibilidade.
  2. Utilizador indica o periodo pretendido e os dados relevantes do animal.
  3. Sistema avalia conflitos com reservas e estadias em curso.
  4. Sistema exclui alojamentos com limpeza por concluir ou incompatibilidade com o animal.
  5. Sistema apresenta os alojamentos disponiveis.
- Fluxo de Excecao:
  - [Nenhum alojamento disponivel] (passo 5)
    1. Sistema informa que nao existem alojamentos disponiveis para os criterios indicados.
