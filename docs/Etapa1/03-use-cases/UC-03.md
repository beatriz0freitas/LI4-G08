# UC-03 - Registar Tutor e Animal

- Use Case: Registar Tutor e Animal
- Descricao: Permite registar um tutor e associar-lhe animais, garantindo que cada animal fica associado a um unico tutor.
- Cenario: Um tutor contacta o hotel pela primeira vez e o funcionario regista os seus dados e os dados do animal para futuras reservas.
- Pre-condicao: Funcionario autenticado.
- Pos-condicao: Tutor e animal registados; animal disponivel para reservas e consultas de historico.
- Fluxo Normal:
  1. Funcionario seleciona a opcao de registar tutor.
  2. Funcionario introduz os dados obrigatorios do tutor.
  3. Sistema regista o tutor.
  4. Funcionario seleciona a opcao de adicionar animal ao tutor.
  5. Funcionario introduz os dados obrigatorios do animal.
  6. Sistema regista o animal associado ao tutor.
- Fluxo Alternativo:
  - [Tutor ja existe] (passo 1)
    1. Funcionario pesquisa o tutor existente.
    2. Funcionario seleciona o tutor.
    3. Fluxo retoma no passo 4.
