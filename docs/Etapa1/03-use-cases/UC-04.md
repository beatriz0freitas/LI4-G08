# UC-04 - Criar Reserva

- Use Case: Criar Reserva
- Descricao: Permite ao Funcionario de Rececao criar uma reserva para um animal, associando-a a um alojamento disponivel no periodo pretendido.
- Cenarios: Tutor contacta o hotel para reservar um alojamento para o seu cao; o funcionario verifica a disponibilidade, cria a reserva e confirma-a.
- Pre-condicao: Funcionario autenticado; tutor e animal registados no sistema; existe pelo menos um alojamento disponivel para o periodo.
- Pos-condicao: Reserva criada em estado `ATIVA`; alojamento fica indisponivel no periodo definido enquanto a reserva estiver ativa.
- Fluxo Normal:
  1. Funcionario seleciona o tutor e o animal associado.
  2. Funcionario indica o periodo pretendido.
  3. <<include>> Consultar Disponibilidade de Alojamento.
  4. Funcionario seleciona o alojamento.
  5. Sistema cria a reserva e marca o alojamento como indisponivel no periodo.
- Fluxo de Excecao:
  - [Nenhum alojamento disponivel] (passo 3)
    1. Sistema informa a indisponibilidade e sugere a consulta de datas alternativas.
