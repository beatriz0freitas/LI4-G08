# UC-04 - Criar Reserva

- Use Case: Criar Reserva
- Descricao: Permite criar uma reserva para um animal, associando tutor, animal, alojamento e periodo, apos validacao de disponibilidade.
- Cenario: Um tutor solicita uma reserva para o seu animal e o funcionario cria a reserva com base nos alojamentos disponiveis.
- Pre-condicao: Funcionario autenticado; tutor e animal registados; periodo pretendido conhecido.
- Pos-condicao: Reserva criada; alojamento considerado indisponivel para o periodo reservado enquanto a reserva se mantiver valida.
- Fluxo Normal:
  1. Funcionario seleciona o tutor.
  2. Funcionario seleciona o animal associado ao tutor.
  3. Funcionario indica o periodo pretendido.
  4. <<include>> Consultar Disponibilidade de Alojamentos.
  5. Funcionario seleciona um alojamento disponivel.
  6. Sistema cria a reserva com tutor, animal, alojamento e periodo.
- Fluxo de Excecao:
  - [Sem alojamento disponivel] (passo 4)
    1. Sistema informa a indisponibilidade.
