# UC-07 - Registar Check-out

- Use Case: Registar Check-out
- Descricao: Permite ao Funcionario de Rececao registar a saida do animal, encerrando a estadia. Se existirem servicos adicionais registados durante a estadia, o pagamento dos mesmos e processado neste momento.
- Cenarios: Tutor chega para levantar o animal; o funcionario regista o check-out; se houver servicos extra, o sistema apresenta o valor e processa o pagamento.
- Pre-condicao: Funcionario autenticado; existe uma estadia em curso com check-in registado.
- Pos-condicao: Estadia encerrada; alojamento marcado como a necessitar de limpeza; se existirem servicos adicionais, pagamento registado.
- Fluxo Normal:
  1. Funcionario localiza a estadia em curso do animal.
  2. Sistema apresenta o resumo da estadia. Nao existem servicos adicionais registados.
  3. Funcionario regista o check-out.
  4. Sistema encerra a estadia e marca o alojamento como a necessitar de limpeza.
- Fluxo Alternativo:
  - [Existem servicos adicionais registados] (passo 2)
    1. Sistema apresenta o resumo da estadia e lista os servicos adicionais com o respetivo valor.
    2. <<include>> Processar Faturacao e Pagamento.
    3. Fluxo retoma no passo 3.
- Origem: RF-09; RF-10; RD-03; RD-04
