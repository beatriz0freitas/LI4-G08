# UC-07 - Registar Check-out

- Use Case: Registar Check-out
- Descricao: Permite registar a saida de um animal, fechar a estadia, cobrar valores complementares e assinalar o alojamento como necessitando de limpeza.
- Cenario: O tutor vem buscar o animal; o funcionario revê a estadia, cobra servicos extra, intervencoes e dias adicionais quando existirem, e encerra a estadia.
- Pre-condicao: Funcionario autenticado; existe uma estadia em curso.
- Pos-condicao: Estadia terminada; reserva concluida; pagamento complementar registado quando aplicavel; alojamento pendente de limpeza.
- Fluxo Normal:
  1. Funcionario pesquisa a estadia em curso.
  2. Sistema apresenta o resumo da estadia.
  3. Sistema verifica a existencia de valores complementares.
  4. Funcionario confirma o check-out.
  5. Sistema termina a estadia e conclui a reserva associada.
  6. Sistema assinala o alojamento como necessitando de limpeza.
- Fluxo de Excecao:
  - [Estadia nao esta em curso] (passo 1)
    1. Sistema informa que o check-out nao pode ser realizado.
    2. Check-out nao e realizado.
- Fluxo Alternativo:
  - [Existem valores complementares] (passo 3)
    1. Sistema apresenta os valores complementares a cobrar.
    2. <<include>> Processar Pagamento.
    3. Fluxo retoma no passo 4.
  - [Passos obrigatorios por completar] (passo 4)
    1. Sistema informa o que falta concluir.
    2. Funcionario regulariza a situacao antes de confirmar o check-out.
    3. Fluxo retoma no passo 4.
