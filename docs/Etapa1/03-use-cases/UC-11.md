# UC-11 - Gerir Historial Clinico

- Use Case: Gerir Historial Clinico
- Descricao: Permite ao medico veterinario consultar o historial clinico e operacional de um animal e registar intervencoes, prescricoes e custos associados durante uma estadia ativa.
- Cenario: O medico e informado de uma alteracao no estado de saude, consulta o historial do animal e regista a intervencao efetuada.
- Pre-condicao: Medico veterinario autenticado; animal registado; para registo de intervencao, animal com estadia ativa.
- Pos-condicao: Historial consultado; intervencao ou prescricao registada; custo associado a estadia quando aplicavel.
- Fluxo Normal:
  1. Medico veterinario pesquisa ou seleciona o animal.
  2. Sistema apresenta o historial clinico e operacional consolidado do animal.
  3. Medico veterinario regista a intervencao ou prescricao, quando aplicavel.
  4. Medico veterinario indica o custo associado, se existir.
  5. Sistema guarda o registo clinico e associa o custo a estadia ativa.
- Fluxo Alternativo:
- Fluxo de Excecao:
  - [Animal sem estadia ativa para intervencao] (passo 3)
    1. Sistema impede o registo de intervencao com custo associado a estadia.
