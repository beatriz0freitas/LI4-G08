# UC-12 - Registar Limpeza de Alojamento

- Use Case: Registar Limpeza de Alojamento
- Descricao: Permite ao responsavel pela limpeza consultar alojamentos pendentes de limpeza e marcar cada alojamento como limpo.
- Cenario: Apos um check-out, o alojamento fica pendente de limpeza; o responsavel limpa-o e atualiza o estado.
- Pre-condicao: Responsavel pela limpeza autenticado; existe pelo menos um alojamento pendente de limpeza.
- Pos-condicao: Alojamento marcado como limpo; pode voltar a ser considerado em consultas de disponibilidade, se nao existir outro impedimento.
- Fluxo Normal:
  1. Responsavel consulta a lista de alojamentos pendentes de limpeza.
  2. Responsavel seleciona o alojamento limpo.
  3. Responsavel confirma a conclusao da limpeza.
  4. Sistema atualiza o estado de limpeza do alojamento para concluido.

- Origem: RF-15; RD-01
