# UC-13 - Consultar Dashboard e Gerar Relatorios

- Use Case: Consultar Dashboard e Gerar Relatorios
- Descricao: Permite ao Diretor consultar indicadores operacionais e financeiros e gerar relatorios filtraveis por periodo, tipo de alojamento, inclusao de servicos extra e agrupamento.
- Cenario: No final de um periodo, a diretora acede ao painel, consulta ocupacao, faturacao e pagamentos pendentes, e exporta um relatorio para apoio a gestao.
- Pre-condicao: Diretor autenticado.
- Pos-condicao: Indicadores consultados; relatorio gerado; exportacao CSV ou PDF valida produzida quando solicitada.
- Fluxo Normal:
  1. Diretor acede a visao operacional.
  2. Sistema apresenta indicadores de ocupacao, reservas, estadias, limpeza, faturacao e pagamentos pendentes.
  3. Diretor define filtros de periodo e, quando aplicavel, tipo de alojamento, agrupamento ou inclusao de servicos extra.
  4. Sistema gera o relatorio com os criterios definidos e aplica o agrupamento na agregacao dos dados.
  5. Diretor consulta ou exporta o relatorio em CSV ou PDF valido.
- Fluxo Alternativo:
  - [Sem dados para os criterios selecionados] (passo 4)
    1. Sistema informa que nao existem dados para os criterios indicados.
    2. Diretor pode alterar os filtros.
    3. Fluxo retoma o passo 4.
- Fluxo de Excecao:
  - [Periodo superior ao limite de exportacao sincronica] (passo 4)
    1. Sistema informa que o periodo maximo para exportacao imediata e 3 meses.
    2. Diretor seleciona um intervalo menor ou solicita processamento posterior quando disponivel.
