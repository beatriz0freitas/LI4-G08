# UC-14 - Gerir Colaboradores e Perfis

- Use Case: Gerir Colaboradores e Perfis
- Descricao: Permite ao Diretor criar, editar e desativar colaboradores, definindo o respetivo perfil de acesso.
- Cenario: A diretora integra um novo colaborador, atribui-lhe o perfil adequado e, quando necessario, atualiza ou desativa a conta.
- Pre-condicao: Diretor autenticado.
- Pos-condicao: Colaborador criado, atualizado ou desativado; operacao registada para auditoria.
- Fluxo Normal:
  1. Diretor acede a gestao de colaboradores.
  2. Sistema apresenta a lista de colaboradores.
  3. Diretor cria ou edita os dados do colaborador e seleciona o perfil.
  4. Sistema valida dados obrigatorios, unicidade de username/email e perfil selecionado.
  5. Sistema guarda o colaborador e regista a operacao de auditoria.
- Fluxo Alternativo:
  - [Desativar colaborador] (passo 3)
    1. Diretor seleciona um colaborador ativo.
    2. Sistema marca a conta como inativa e regista a operacao de auditoria.
- Fluxo de Excecao:
  - [Dados invalidos ou duplicados] (passo 4)
    1. Sistema rejeita a operacao e apresenta a causa.
- Origem: RF-02; RNF-04; RNF-09
