# UC-16 - Consultar Auditoria

- Use Case: Consultar Auditoria
- Descricao: Permite ao Diretor consultar e exportar eventos de auditoria associados a operacoes criticas.
- Cenario: A diretora quer perceber quem realizou uma operacao critica, filtrando por periodo, utilizador, operacao ou entidade.
- Pre-condicao: Diretor autenticado.
- Pos-condicao: Eventos de auditoria consultados ou exportados em CSV.
- Fluxo Normal:
  1. Diretor acede a auditoria.
  2. Sistema apresenta filtros de consulta.
  3. Diretor define periodo e filtros opcionais.
  4. Sistema consulta os eventos de auditoria persistidos.
  5. Sistema apresenta os resultados paginados.
- Fluxo Alternativo:
  - [Exportar CSV] (apos passo 5)
    1. Diretor solicita exportacao.
    2. Sistema gera CSV com os eventos filtrados.
- Fluxo Alternativo:
  - [Periodo omitido ou invertido] (passo 3)
    1. Sistema aplica periodo predefinido ou normaliza a ordem das datas.
    2. Sistema continua a consulta com o periodo normalizado.
- Origem: RF-19; RNF-08; RNF-09
