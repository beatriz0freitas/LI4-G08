' ==================== NOTES ====================
note right of Alojamento
RD-01: Disponibilidade depende de
periodo, especie e estado de limpeza
end note

note right of Reserva
RD-02: Uma reserva confirmada pode
originar no máximo uma estadia
end note

note right of Estadia
RD-03: Uma estadia exige
exatamente uma fatura
end note

note right of Fatura
RD-04: A fatura suporta relatório
financeiro e auditoria por tutor
end note

note right of Animal
RD-05: O estado de saúde deve
ser um enum fechado
end note

note right of IntervencaoClinica
RD-06: A intervenção clínica pode
originar um serviço extra opcional
end note

note right of Cuidado
RD-07: Os cuidados diários são
registados pela estadia, não pelo controller
end note

note right of AlojamentoStateListener
RD-08: Mudanças de estado de alojamento
notificam limpeza (Observer pattern)
end note

note right of ReservaRepository
RD-09: Consultas por período e estado
são obrigatórias para rastreabilidade
end note
