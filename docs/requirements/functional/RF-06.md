# RF-06 - Controlo de disponibilidade de boxes

**Descrição:**
O sistema deve determinar a disponibilidade de cada box em tempo real com base em três condições cumulativas: (1) inexistência de reserva confirmada para o período pretendido; (2) inexistência de estadia ativa nesse período; e (3) estado de limpeza registado como "concluído". Sempre que uma destas condições deixe de se verificar, a box deve ser automaticamente marcada como indisponível. O sistema deve impedir a criação de reservas que violem estas condições e apresentar alternativas válidas.

**Origem:** US-06, US-12, US-21

**Prioridade:** Must Have
